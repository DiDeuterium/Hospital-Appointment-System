### BF-03 挂号预约 + 号源扣减（模拟事务）
**SQL：**
```sql
START TRANSACTION;
INSERT INTO appointment (patient_id, schedule_id, status, create_time)
VALUES (6, 17, 1, NOW());
SET @new_appt_id = LAST_INSERT_ID();

UPDATE schedule
SET rest_quota = rest_quota - 1
WHERE schedule_id = 17 AND rest_quota > 0;

SELECT rest_quota FROM schedule WHERE schedule_id = 17;

-- 取消挂号并回补号源
UPDATE appointment SET status = 2 WHERE appt_id = @new_appt_id;
UPDATE schedule SET rest_quota = rest_quota + 1 WHERE schedule_id = 17;

SELECT status FROM appointment WHERE appt_id = @new_appt_id;
SELECT rest_quota FROM schedule WHERE schedule_id = 17;
ROLLBACK;
```
**预期结果：**
- 扣减后 `schedule_id=17` 的 `rest_quota` 从 45 变为 44；
- 取消后 `status=2`，`rest_quota` 回到 45。  
**实际结果：** 待执行记录。

## 三、查询测试（QT）

### QT-01 多表连接：按科室+日期查排班与医生
**SQL：**
```sql
SELECT
    s.schedule_id,
    s.work_date,
    s.shift,
    s.total_quota,
    s.rest_quota,
    d.doc_id,
    d.doc_name,
    dp.dept_name
FROM schedule s
JOIN doctor d     ON s.doc_id = d.doc_id
JOIN department dp ON d.dept_id = dp.dept_id
WHERE dp.dept_id = 1
  AND s.work_date = '2026-05-20'
ORDER BY s.shift, d.doc_id;
```
